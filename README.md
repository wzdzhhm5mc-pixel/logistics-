# # Mirage Logistics

Mirage Logistics is a premium enterprise logistics platform built with modern web technologies.

## Features

- Global shipment tracking
- Instant freight quotation
- Customer dashboard
- Admin dashboard
- Warehouse management
- AI logistics assistant
- Partner marketplace
- Billing & payments
- Realtime notifications
- Interactive logistics map

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Supabase
- Mapbox
